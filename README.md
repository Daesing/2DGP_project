## 🎮 Control Keys
- **a key**: Slash  
- **d key**: Dash  
- **s key**: Focus (health recovery)  
- **q key**: Fireball (requires skill points)  
- **Arrow keys**: Move  
- **Up arrow + a key**: Upslash  
- **Down arrow + a key**: Downslash  
- **Spacebar**: Jump
